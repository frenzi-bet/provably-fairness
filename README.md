# provably-fairness
## Verification method
### Local running verification
1.Ensure that Python 3 is installed on your computer.
2.Download the code to your local machine.
3.Cd to the code directory
4.Execute python3 provably-fairness.py client_seed server_seed nonce
### Online compilation verification
1.Copy the code content of provably-fairness.py to the Google Colab online compiler.
2.Note to modify the client_seed, server_seed, and nonce at the end of the document to your own content.
3.Click “Run”.
